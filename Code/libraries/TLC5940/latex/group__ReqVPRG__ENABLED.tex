Sets the dot correction data from an array in progmem. 

An example\+: 
\begin{DoxyCode}
1 #include "tlc\_progmem\_utils.h"
2 prog\_uint8\_t dcArray1[NUM\_TLCS * 12] = \{
3   DC\_QUARTET(32, 63, 32, 63), DC\_QUARTET(32, 63, 32, 63),
4   DC\_QUARTET(32, 63, 32, 63), DC\_QUARTET(32, 63, 32, 63),
5 \};
6 
7 // sometime after Tlc.init()
8 tlc\_setDCfromProgmem(dcArray1);
\end{DoxyCode}
 This would set every other channel to have a dot correction value of 32. (Although the N\+U\+M\+\_\+\+T\+L\+CS $\ast$ 12 looks like an error, each \hyperlink{tlc__config_8h_ab4f8107b4cdf76352f7cd594d2370cf9}{D\+C\+\_\+\+Q\+U\+A\+R\+T\+ET} is 3 bytes). The array would have to be expanded if \hyperlink{tlc__config_8h_aa673c2d8e04c9cca25f51d6c2b516b8f}{N\+U\+M\+\_\+\+T\+L\+CS} != 1.

The Format of the array is similar to \hyperlink{Tlc5940_8h_a1560009c42233131f7abe72e8113f793}{tlc\+\_\+\+G\+S\+Data}, the last channel of the last T\+LC is the first value in the array. In the example above, the first 32 is setting O\+U\+T15, and the last 63 is setting O\+U\+T0.
